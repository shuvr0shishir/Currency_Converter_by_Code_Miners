# Currency Converter (C Project)

This is a simple Currency Converter developed using the C programming language.

The program converts currencies using predefined exchange rates stored within the system. Since it does not use real-time API data, the conversion results are approximate and based on the stored values.

This project was created as part of an academic team project to practice programming logic, problem-solving, and fundamental concepts of C programming.

## Features
- Convert between multiple currencies
- Simple command-line interface
- Built using C programming language

## Purpose
This project was developed as part of a university project contest.
